自定义页面示例

把 HTML 文件放到 ui/functions/，更多功能页会自动新增入口。

支持格式
文件名任意安全的 *.html
入口名优先读取 <title>
页面内容可写完整 HTML，也可只写片段
动态值支持现有 {{TOKEN}} 替换
自动外壳尺寸
屏幕320 x 480 px
状态栏顶部约 26 px，由 DevUI 自动绘制
返回栏42 px，高度固定，自动插入
内容区从返回栏下方开始，可上下滚动
推荐写法

优先复用 card、title、row、kv-l、val 这些现有样式。不要依赖 JavaScript、图片或 CSS Grid。

触控触发方式
可点元素使用 <a href="act:...">
触发时机手指点击并松开后执行
点击范围按元素排版后的矩形区域识别
不要使用JS、onclick、button、form

整张卡可点时，把 card 写到 <a> 上；小按钮也必须是 <a>。

示例：切换深浅色
点击整张卡触发 act:theme
常用 act 动作
返回act:backfunc
打开子页act:sub:wifi.html
打开自定义页act:func:example.html
信令设置act:openmodal:sig
重新搜网act:sigrescan
返回 WiFi 短信 信令
